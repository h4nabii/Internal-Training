# JavaScript 中的基础变量类型

## 基础类型

JavaScript 中的基础类型（Primitive Types）包括：

- `string`：字符串类型，用于表示文本数据
- `number`：数字类型，用于表示整数和浮点数
- `boolean`：布尔类型，只有两个值 `true` 和 `false`
- `null`：表示“无”或“空”的值，只有一个值 `null`
- `undefined`：表示变量未定义或未赋值，只有一个值 `undefined`
- `symbol`：符号类型，表示独一无二的值，常用于对象属性的唯一标识
- `bigint`：大整数类型，用于表示超过 `Number.MAX_SAFE_INTEGER` 的整数

基础类型的值是不可变的（immutable），即一旦创建就不能修改。对基础类型的操作会返回一个新的值，而不是修改原有的值。此处的“不可变”指的是值本身，而不是变量绑定的不可变。基础类型的变量可以重新赋值，但原有的值不会被修改。

```js
let str = "Hello";
str[0] = "h"; // 试图修改字符串的第一个字符
console.log(str); // "Hello"，字符串没有被修改
```

获取一个变量的类型可以使用 `typeof` 操作符：

```js
console.log(typeof "Hello"); // "string"
console.log(typeof 42); // "number"
console.log(typeof true); // "boolean"
console.log(typeof null); // "object"（这是 JavaScript 的一个历史遗留问题）
console.log(typeof undefined); // "undefined"
console.log(typeof Symbol("sym")); // "symbol"
console.log(typeof 9007199254740991n); // "bigint"
```
