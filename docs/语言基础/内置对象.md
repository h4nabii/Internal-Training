# 内置对象

本章聚焦 JavaScript 标准内置对象的使用边界与选型思路，帮助你从“会查 API”进阶到“会选工具”。

## 学习目标

- 理解内置对象的定义、分类和适用场景
- 区分原始值与包装对象，避免隐式装箱带来的误解
- 掌握常见内置对象（如 `Object`、`Array`、`Map`、`Set`、`Promise`）的核心职责
- 能按“遍历/转换/校验/时间/错误处理”等维度组织常用方法
- 识别高频坑点并形成可维护实践

## 1. 什么是内置对象

### 1.1 定义

- 由 ECMAScript 标准提供、运行时默认可用的对象或构造器
- 典型示例：`Object`、`Array`、`Date`、`RegExp`、`Promise`

### 1.2 内置对象 vs 宿主对象

- 内置对象：语言标准定义
- 宿主对象：运行环境提供（如浏览器的 `window`、Node.js 的 `process`）

### 1.3 分类视角

- 值包装与基础能力：`Object`、`String`、`Number`、`Boolean`
- 集合与结构：`Array`、`Map`、`Set`
- 时间与数学：`Date`、`Math`
- 文本规则与序列化：`RegExp`、`JSON`
- 错误与异步：`Error`、`Promise`

## 2. 原始值与包装对象

### 2.1 自动装箱与拆箱

- 原始值调用方法时会发生临时装箱
- 例如字符串可直接调用 `toUpperCase()`

### 2.2 常见误区

- 不建议手动创建包装对象（如 `new String("x")`）
- 包装对象是对象，参与比较时行为容易让人困惑

## 3. 常见内置对象大纲

### 3.1 `Object`

- 核心职责：对象创建、属性操作、原型相关能力
- 重点方法：`keys`、`values`、`entries`、`assign`、`hasOwn`
- 注意点：浅拷贝与原型污染风险

### 3.2 `Array`

- 核心职责：有序数据存储与批量处理
- 重点方法：`map`、`filter`、`reduce`、`find`、`some`、`every`
- 注意点：区分“是否修改原数组”

### 3.3 `String` / `Number`

- 核心职责：文本和数值处理
- 重点方法：`trim`、`slice`、`includes`、`toFixed`
- 注意点：浮点精度问题与 `NaN` 判断

### 3.4 `Math` / `Date`

- `Math`：数学运算工具集合（无实例）
- `Date`：日期时间读写与格式化基础
- 注意点：时区、时间戳单位（秒 vs 毫秒）

### 3.5 `RegExp` / `JSON`

- `RegExp`：模式匹配与文本提取
- `JSON`：序列化与反序列化
- 注意点：`JSON.stringify` 会丢失函数、`undefined`、`Symbol`

### 3.6 `Map` / `Set`

- `Map`：键值对集合，键可为任意类型
- `Set`：值唯一集合，常用于去重
- 注意点：和普通对象在键类型、遍历语义上的差异

### 3.7 `Error` / `Promise`

- `Error`：统一错误信息结构与堆栈
- `Promise`：异步状态管理与链式处理
- 注意点：错误传播链路与未处理拒绝

## 4. 方法分组速查

- 创建与转换：`Array.from`、`Object.fromEntries`、`Number()`
- 遍历与变换：`map`、`filter`、`reduce`
- 查找与判断：`find`、`includes`、`has`、`hasOwn`
- 聚合与计算：`reduce`、`Math.max/min`
- 序列化与反序列化：`JSON.stringify/parse`
- 异步控制：`Promise.all/allSettled/race/any`

## 5. 常见坑清单

- `typeof null === "object"`（历史遗留）
- `new Number(1) !== 1`（包装对象与原始值混淆）
- 浅拷贝导致嵌套对象联动修改
- `Date` 解析格式不统一导致跨环境差异
- 正则贪婪匹配与 `g` 标志状态污染
- `JSON.parse` 未做异常处理导致运行时崩溃

## 6. 性能与实践建议

- 先保证可读性，再做性能优化
- 大量键值操作优先评估 `Map`，去重优先评估 `Set`
- 对外部输入做类型校验，不信任 JSON 字段
- 错误对象统一封装，便于日志与监控系统消费
- 对日期和时区逻辑建立统一工具层，避免散落处理

## 7. 练习题建议

### 7.1 基础题

- 判断下列表达式类型与结果（含包装对象）
- 用 `Array` 方法完成筛选、映射、求和

### 7.2 进阶题

- 用 `Map` 重构对象字典，比较可读性与行为差异
- 用 `Set` 实现数组去重并保留插入顺序

### 7.3 综合题

- 解析一段 JSON 数据并完成清洗、校验、聚合
- 设计一段异步数据处理流程，要求统一错误处理
