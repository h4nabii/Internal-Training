# 常见面试题

## 手写防抖和节流

**防抖（Debounce）** 是一种优化技术，用于限制函数的执行频率。当一个事件被触发时，防抖会等待一段时间，如果在这段时间内再次触发事件，则重新计时。只有当事件停止触发超过指定时间后，函数才会被执行。

::: code-group

```js [index.js]
// 防抖（Debounce）
function debounce(func, wait) {
  let timeout = null;

  return function (...args) {
    if (timeout !== null) {
      clearTimeout(timeout);
    }
    timeout = setTimeout(() => {
      func.apply(this, args);
    }, wait);
  };
}
```

```ts [index.ts]
// 防抖（Debounce）
function debounce(func: Function, wait: number) {
  let timeout: number | null = null;

  return function (...args: any[]) {
    if (timeout !== null) {
      clearTimeout(timeout);
    }
    timeout = setTimeout(() => {
      func.apply(this, args);
    }, wait);
  };
}
```

:::

**节流（Throttle）** 也是一种优化技术，用于限制函数的执行频率。当一个事件被触发时，节流会立即执行函数，并且在指定的时间间隔内禁止再次执行。只有当时间间隔结束后，函数才会再次被允许执行。

::: code-group

```js [index.js]
// 节流（Throttle）
function throttle(func, wait) {
  let lastTime = 0;

  return function (...args) {
    const now = Date.now();
    if (now - lastTime >= wait) {
      func.apply(this, args);
      lastTime = now;
    }
  };
}
```

```ts [index.ts]
// 节流（Throttle）
function throttle(func: Function, wait: number) {
  let lastTime = 0;

  return function (...args: any[]) {
    const now = Date.now();
    if (now - lastTime >= wait) {
      func.apply(this, args);
      lastTime = now;
    }
  };
}
```
