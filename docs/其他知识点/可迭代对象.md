# 可迭代对象

详情可参考 MDN 文档：[迭代协议](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Iteration_protocols)

可迭代对象（Iterable）是指实现了迭代协议的对象，能够被 `for...of` 循环、扩展运算符等语法结构遍历。

常见的可迭代对象包括：数组、字符串、`Map`、`Set`、`arguments` 对象等。

可迭代对象必须实现一个特殊的方法 `Symbol.iterator`，该方法返回一个迭代器对象。迭代器对象必须实现一个 `next()` 方法，每次调用 `next()` 都返回一个包含 `value` 和 `done` 属性的对象。

```ts
function createDownCounter(n: number) {
  return {
    [Symbol.iterator]() {
      return {
        next() {
          if (n > 0) {
            // 返回当前值并递减
            return { value: n--, done: false };
          } else {
            // 计数结束，标记迭代完成
            return { value: undefined, done: true };
          }
        },
      };
    },
  };
}

const counter = createDownCounter(3);

for (const num of counter) {
  console.log(num); // 3, 2, 1
}
```

## 生成器

生成器（Generator）是一种特殊的函数，可以暂停和恢复执行，使用 `function*` 语法定义。生成器函数返回一个迭代器对象，可以通过 `yield` 关键字逐步生成值。

```ts
function* countUpTo(n: number) {
  for (let i = 1; i <= n; i++) {
    yield i; // 暂停并返回当前值
  }
}

const counter = countUpTo(3);
const val1 = counter.next(); // { value: 1, done: false }
const val2 = counter.next(); // { value: 2, done: false }
const val3 = counter.next(); // { value: 3, done: false }
const val4 = counter.next(); // { value: undefined, done: true }
```

不难发现，生成器可以生成一个迭代器对象，而可迭代对象则需要实现 `Symbol.iterator` 方法返回一个迭代器对象。生成器函数的语法糖使得编写迭代器变得更加简洁和直观。

```ts
class Person {
  name: string;
  age: number;

  constructor(name: string, age: number) {
    this.name = name;
    this.age = age;
  }

  *[Symbol.iterator]() {
    yield this.name;
    yield this.age;
  }
}

const p = new Person("Alice", 30);

for (const prop of p) {
  console.log(prop); // "Alice", 30
}
```
