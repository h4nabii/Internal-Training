# JavaScript 中的类

> 类这个概念在大部分面向对象编程语言中都是一个重要的组成部分，JavaScript 也不例外。虽然 JavaScript 在 ES6 之前没有正式的类定义，但开发者可以通过函数和原型链来模拟类的行为。ES6 引入了 `class` 关键字，使得创建类和实现面向对象编程变得更加直观和简洁。

## 什么是类（Class）

### 定义

类是用于创建相同类型对象的 **模板** 或 **蓝图**

类的名称来源于 **分类** 或 **类别** 的概念。它定义了一类对象的属性和方法，允许我们通过实例化来创建具有相同结构和行为的对象。

### 展开解释

类定义了一类对象有什么数据（属性）和能做什么操作（方法）。

通过同一个类，可以生产出无数个结构相同但数据不同的对象（实例）。

## 如何定义类

定义一个类需要用到 `class` 关键字，后面跟上类的名称和一对花括号 `{}`。在花括号内，我们可以定义构造函数（constructor）和其他方法。

::: code-group

```js [Person.js]
class Person {
  constructor(name, age) {
    // 定义属性并赋值
    this.name = name;
    this.age = age;
    this.score = 0; // 可以在 constructor 中为属性赋默认值
  }

  greet() {
    // 定义方法
    console.log(`你好，我是 ${this.name}，我今年 ${this.age} 岁。`);
  }
}
```

```ts [Person.ts]
class Person {
  // TS 需要先声明属性的类型
  name: string;
  age: number;
  score: number;

  constructor(name: string, age: number) {
    // 为属性赋值
    this.name = name;
    this.age = age;
    this.score = 0; // 可以在 constructor 中为属性赋默认值
  }

  greet() {
    // 定义方法
    console.log(`你好，我是 ${this.name}，我今年 ${this.age} 岁。`);
  }
}
```

:::

## 如何使用类

定义好类之后，我们可以通过 `new` 关键字来创建类的实例（对象）。实例化时会调用类的构造函数来初始化对象的属性。

```js
const person1 = new Person("Alice", 30);
person1.greet(); // 你好，我是 Alice，我今年 30 岁
console.log(person1.score); // 0
console.log(person1 instanceof Person); // true
console.log(person1.constructor); // [class Person]，constructor 属性指向类本身
```
